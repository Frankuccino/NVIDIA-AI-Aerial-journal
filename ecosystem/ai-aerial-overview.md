# NVIDIA Aerial Framework

**Purpose:**  
The Aerial Framework provides a **software layer** for deploying AI-native workloads on telecom networks, with GPU acceleration.

**Key Features:**

- Tools for **AI inference in RAN pipelines**
- Supports **CUDA-accelerated signal processing**
- Works alongside **Sionna** for simulation
- Integration with **Omniverse Digital Twin** for network testing

**Why It Matters:**  
It allows operators and researchers to **experiment with AI directly inside network infrastructure**, shortening development cycles and improving performance.
