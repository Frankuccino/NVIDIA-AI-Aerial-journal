# CUDA-Accelerated RAN

**Purpose:**  
A GPU-accelerated **Radio Access Network (RAN)** stack for AI-native 5G → 6G networks.

**Key Features:**

- GPU replaces traditional ASIC/FPGA for baseband processing
- Enables **real-time AI-driven optimization**
- Reduces latency for network control loops
- Works with the **Aerial Framework** and **Sionna**

**Why It Matters:**  
Traditional RAN hardware is rigid. By moving processing to GPUs:

- Networks become **software-defined**
- AI can control the network in **real-time**
- Rapid experimentation is possible
